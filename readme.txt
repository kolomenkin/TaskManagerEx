=============================================================================
== TaskManagerEx.dsw contains 4 projects:
=============================================================================

AccessMaster		- modified tool from Jeffrey Richter, it contains
			some shared files for TaskManagerExDll
TaskManagerEx		- Task Manager Extension loader
TaskManagerExDll	- Task Manager Extension hook dll
TaskManagerExTest	- test environment for TaskManagerExDll. It designed
			to debug TaskManagerEx.dll
